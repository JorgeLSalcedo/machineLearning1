Georefence script for any text

The purpose of this script is geolocalize news that previously have been not tagged on any place. We develop a script that identify all the words that begin with capitalized letters.

For instances, proper personal names "Carlos, Laia, Joan", as well as places: "Barcelona, New York, Notre Dam, Eiffel Tower , etc". The script creates a library of all the capitalizes word and in this list of word, the script cheks if the word can be geolocalized.

How the script geolocalizes.

The script use wikipedia API, each time it finds a new word that begin with capital letter, it connects with Wikipedia and chekcks if the word can be geolocalized. If the word has not been geolocalized the program save the pale in the library with its coordinates. In this way the search for the next time its going to be more efficient because if the word is on the library we have to to connect againg with wikipedia.

Why wikipedia and not a geolocalization library.

Wkipedia give us the chance to develop a personalize tools for almost any type of language. In this first beta version the courpus of news (text) is in catala, but we can adapt relatevy easy or script to courpus on any languagues.
